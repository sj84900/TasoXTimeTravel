## 🚀 TasoXTimeTravel

### 🛠 주요 기능

1. **날짜 이동 버튼 (검색 페이지)**
* 우측 하단에 검색 결과 화면에서 검색 기간(`since`, `until`)을 **하루씩 앞뒤로 이동**할 수 있는 화살표 버튼이 생깁니다.
* 주소창을 직접 수정하지 않고 클릭만으로 날짜별 검색 결과를 볼 수 있습니다.


2. **6년 전 오늘 검색 (일반 페이지)**
* 검색 페이지가 아닐 때는 우측 하단에 돋보기 버튼이 나타납니다.
* 클릭 한 번으로 지정된 계정(`@amanekanatach`)의 **정확히 6년 전 오늘 게시물**을 바로 찾아줍니다.
